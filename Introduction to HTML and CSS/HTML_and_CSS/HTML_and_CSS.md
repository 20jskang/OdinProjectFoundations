# HTML and CSS

**HTML (Hyper Text Markup Language)** and **CSS (Cascading Style Sheets)** are two languages that work together to create everything that you see when you look at something on the web. HTML is the raw data that a webpage is built out of. All the text, links, cards, lists, and buttons are created in HTML. CSS is what adds styles to those plain elements. HTML puts information on a webpage, and CSS positions that information, give it colour, changes the font, and makes it look great!
